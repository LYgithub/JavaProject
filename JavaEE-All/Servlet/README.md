# javaWeb 三大组件

## Servlet
> `处理`客户端发送的请求  
> 在 web.xml 中进行配置
> 
- 主要任务  
    1、接收请求数据  
    2、处理请求 => Service 、doGet、doPost...  
    3、完成响应  

## Filter
> `拦截｜放行` 客户端发送的请求  
> 多个拦截器的执行顺序，根据web.xml 配置的顺序执行
